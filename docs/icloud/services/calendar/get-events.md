*Get Events*
----
  Send a [request](../../definitions/requests/default-request.md) to fetch the events within a given date range from iCloud calendar.

* **Endpoint**
  
  [Calendar-Endpoint](../../definitions/icloud/endpoints/calendar.md)
  
* **Method:**

  `GET`
  
* **Path**

  `/ca/startup`
  
* **Query Parameters**
 
   `clientBuildNumber=1923Hotfix2`
   
   `clientMasteringNumber=1923Hotfix2`
   
   `clientId=[HMAC]`
   
   `dsid=[dsid]`
   
   `clientVersion=5.1`
   
   `lang=[lang]`
   
   `usertz=[usertz]`
   
   `startDate=[startDate]`
   
   `endDate=[endDate]`
   
   You need to pass the dynamic parameters [[HMAC]](../../definitions/general/types/hmac.md), [[dsid]](../../definitions/icloud/variables/session-id.md), [[lang]](../../definitions/general/types/locale.md), [[usertz]](../../definitions/general/types/timezone.md), [[startDate]](../../definitions/general/types/date.md) and [[endDate]](../../definitions/general/types/date.md).

* **Payload**

  `Empty`
