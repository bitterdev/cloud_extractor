**Validate Session**
----
  Send a request to check if the current session is valid.

* **Method:**

  `GET`
  
* **Path**

  `/account/validate_session`
  
* **Query Parameters**
   
   `clientId`

* **Success Response:**
    
    ```json
    {
       "success":true,
       "data":null,
       "error":null
    }
    ```
